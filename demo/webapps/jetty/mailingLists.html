Jetty Mailing Lists

Hosted by Sourceforge:

    Jetty Announce
     Low volume mailing list for announcements of new releases and developments. The list archives can be view and searched or the list read as a newsgroup at http://news.gmane.org/gmane.comp.java.jetty.announce

    Jetty Discuss
     discussion of Jetty development, integration and issues. The list archives can be view and searched or the list read as a newsgroup at http://news.gmane.org/gmane.comp.java.jetty.general

    Jetty Support
     Mailing list for usage, help and configuration questions. The list archives can be view and searched or the list read as a newsgroup at http://news.gmane.org/gmane.comp.java.jetty.support

    Jetty CVS Log
     The CVS checkin logs are posted to this mailing list.

These lists were moved from yahoogroups in september 2003. Partial archives are still held there
