Jetty Projects

One of the key design aims of the Jetty is to be very flexible and embeddable, so that it can be used stand alone, or embedded within other projects or products. As such, many extentions and embeddings of Jetty have developed. Some of these projects are managed as part of the Jetty project, some are collaborations with other projects, while many more are totally independant (see the Jetty Powered page).

    Jetty:// -The core Jetty project providing HTTP and J2EE servlet container.
    JettyPlus:// -More J2EE integration, such as JNDI, datasources, JTA, JAAS, log4j etc.
    JettyJBoss:// -Integration with the JBoss application server.
    JettyJOnAS:// -Integration with the JOnAS application server.
    JettyPowered:// -Many independant projects and products are powered by Jetty.
